# Oulf · أُلف — Pâtisserie Artisanale

Site e-commerce frontend pour **Oulf**, pâtisserie artisanale spécialisée dans les mini-cakes et douceurs orientales, basée à **Relizane, Algérie**.

## 🌸 Stack Technique

- **Frontend**: React 18 + Vite
- **Routing**: React Router DOM v6
- **Styling**: CSS custom (design system Rose Luxueux)
- **Backend**: *(à venir — Python/FastAPI)*

## 🚀 Démarrage

```bash
npm install
npm run dev
```

## 📁 Structure

```
src/
├── components/   # Navbar, Hero, Products, About, Delivery, Footer
├── context/      # LanguageContext (FR/AR + RTL)
├── data/         # Produits & helpers WhatsApp
└── styles/       # Design system CSS
```

## 🌍 Fonctionnalités

- Langue **Français / العربية** avec RTL automatique
- Commandes via **WhatsApp** → `0668545371`

- Design **Rose Luxueux** (Cormorant Garamond + DM Sans)
